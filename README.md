# DOMINIUM QnA!!!

### - Q: When will development start on the 0.7 version?
### - A: Currently I'll take some time off development on Dominium, and I'll only patch bugs and gather some more ideas (I already have most written down), as I want to work on other projects too.

#

### - Q: How long will it take for the next update to come out?
### - A: It depends on multiple factors: How fast I get those finishing ideas; How ambitious these ideas will be; How much free time I'll have to work on the mod; some other factors that I can't currently think of.

#

### - Q: What can I expect from the Future of Dominium?
### - A: I've pretty much exhausted the publicly available [Charter](https://modrinth.com/mod/charter)/[Mason Decor](https://www.curseforge.com/minecraft/mc-mods/mason-decor) content, so from Dominium 0.8 and so on, you can expect way more original content. Also Dominium 0.7 will be a pretty important release for me as it'll reach the state that I originally wanted to release it in, before rushing things.

#

### - Q: How do you obtain Dominic Orbs/Dominic Weapons in Survival?
### - A: Dominic Orbs & Weapons aren't meant to be obtained in Survival, as the power to Erase players from the World is rather op no matter what. Instead Dominic Orbs or Weapons are meant to be given from either Operators/Deity/Merchant Accounts. The Dominic part of the mod is meant to create roleplay. Also I'm sure I added Dominic Orbs as drops to Withers if you turn on a config, but I don't recommend that option.

#

### - Q: Will my suggestions get added to the Mod?
### - A: It depends. Some features from the current version (0.6) were suggested by some people (ty <3), like the Dominic Bricks Build Set or The QoL Vessel & Obelisk Features. Also most (around 80%) of 0.7 content is already planned out, so most suggestions will be left out for future versions.

#

### - Q: Where can I contact you?
### - A: You can create a question here on GitHub. I prefer to keep all communication here.
